通过 HTTP 对酷 Q 的事件进行上报以及接收 HTTP 请求来调用酷 Q 的 C++ 接口，从而可以使用其它语言编写酷 Q 插件。

此分支是正在编写的新版本。

除了 `README.md` 为 UTF-8 编码，其它代码文件和 JSON 文件均为 GBK 编码。
